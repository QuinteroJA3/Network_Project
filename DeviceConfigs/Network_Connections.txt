Introduction to Computer Networks Project
Network Connection Summary

Topology:
NAT2
  |
Switch1
  |-- Ubuntu_Server-1
  |-- Win10Client-1

Device Roles:

NAT2:
Provides NAT and external network connectivity.

Switch1:
Provides Ethernet communication between all network devices.

Ubuntu_Server-1:
Provides DHCP, DNS, and HTTP services.

Win10Client-1:
Receives its IP configuration through DHCP and is used to test DNS,
HTTP, ICMP, ARP, TCP, and UDP communication.

Protocols Demonstrated:
- DHCP
- DNS
- ARP
- HTTP
- ICMP
- NAT
- TCP
- UDP

Testing and Verification:
Wireshark was used to capture and examine traffic generated by the
Windows 10 client and Ubuntu server. Connectivity and services were
verified using ipconfig, ping, nslookup, curl, and Wireshark filters.
