# Networks Project

## Overview
This project demonstrates the design, configuration, and testing of a small virtual computer network using GNS3. The network includes multiple devices and services configured to demonstrate common networking protocols.

## Technologies
- GNS3
- Cisco IOS Routers
- Ethernet Switches
- Windows 10 Virtual Machine
- Wireshark

## Protocols Demonstrated
- DHCP
- DNS
- ARP
- HTTP
- ICMP
- NAT
- TCP
- UDP

## Repository Structure
- Documentation – Final report and project documentation
- DeviceConfigurations – Router and server configuration files
- GNS3Project – GNS3 project files
- Screenshots – Evidence of successful configuration and testing
- WiresharkCaptures – Packet capture files

## Project Outcome
The completed network successfully provided IP addressing, DNS resolution, web connectivity, and communication between devices. Wireshark was used to verify network traffic and demonstrate the operation of key networking protocols.
